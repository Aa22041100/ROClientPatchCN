#
[公会建立]
 1. 公会建立必须拥有华丽金属.  
 (必须从任务/怪兽这里获得一个华丽金属.)
 
 2. 建立公会的方法: 
 /guild (guild) 
 '建立公会按钮[ALT+G].
 
 3. ALT+G: 公会信息调用. 
#
 
[公会信息窗口]
#
 1. 公会信息:
 公会经验, 公会等级, 公会徽章, 会长姓名, 敌对,
 积分, 平均等级, 联盟/敌对公会都可以查到. 
#
 
 2. 会员信息: 
 公会成员的个人信息
 如果会员退出公会. 则会删除
 会长可以解散公会的所有成员. 
#
 
 3. 职位设置: 
 最大缴税比例50%
#
 
 4. 公会技能:
 公会会长操作.
 如果您的公会升级后拥有了新的技能点.
 可以立即升级公会技能来提高公会实力.
#
 
 5. 驱逐: 
 公会的驱逐者名单
 还包括被驱逐的原因...
#
 
 6. 会长公告:
 会员登录时可以收到通知. 
#
 
[注册和标志]
#
 1) 24x24像素的256色位图(BMP)
 创建徽章.
#
 
 2) 在仙境传说Online的安装目录中
 'Emblem'文件夹中导入徽章. 
#
 
 3) 信息窗口徽章
 点击'编辑'以改变公会徽章. 
#
 
 4) 徽章改变
 公会随时更换徽章, 必须会长操作.
